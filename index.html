1 What are Binary Numbers?

Watch this short video (5 minutes). It will explain what binary numbers are.

Quick recap

  • Computers only understand two values: 0 and 1.
  • Each 0 or 1 is called a bit.
  • 8 bits together are called a byte.
  • Each position has a value: 1, 2, 4, 8, 16, 32, 64, 128.

3 Binary to Denary

Now we go the other way: from binary to a normal (denary) number.

The trick

Whenever you see a 1, add the place value above it. Ignore the 0s.

Example: 0 0 0 0 1 0 1 1 → 8 + 2 + 1 = 11
