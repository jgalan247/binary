# Binary - Year 7 Data Representation

A static one-page lesson site introducing **binary numbers** to Year 7 (low-ability) students.

## What's inside

1. Intro video (5 min) - *What are Binary Numbers?*
2. Interactive 8-bit demo - click bits to see binary &harr; denary live.
3. **Binary Bonanza** game (denary &rarr; binary). Reward: first 5 students to reach the highest level get a "4" Achievement Point.
4. Binary &rarr; denary explainer + interactive working-out demo.
5. **Cisco Binary Game** (mixed practice).
6. Mini test:
   - 4 binary &rarr; denary questions
   - 4 denary &rarr; binary questions
   - 2 multiple-choice questions
   - Live green/red feedback and total score out of 10.

## Run locally

Open `index.html` directly in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Deploy (GitHub Pages)

In the repo settings, enable Pages from the `main` branch root.

## Files

- `index.html` - lesson page
- `css/styles.css` - styling
- `js/app.js` - interactive demos + quiz logic
